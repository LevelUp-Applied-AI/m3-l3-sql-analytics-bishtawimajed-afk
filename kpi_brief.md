# KPI Brief: Levant Tech Solutions

## 1. Department Salary Efficiency
* **Name:** Average Salary per Department
* **Definition:** Calculated using Query Q5, which averages employee salaries grouped by department.
* **Current Value:** $58,000 (Example based on Q5 results).
* **Interpretation:** This shows how much, on average, each department costs in terms of human resources, helping in budget planning.

## 2. Project Staffing Ratio
* **Name:** Average Employees per Project
* **Definition:** Calculated using Query Q4 by counting assigned employees for each project.
* **Current Value:** 2.5 Employees (Example based on Q4 results).
* **Interpretation:** Helps management understand if projects are understaffed or if resources are distributed effectively.

## 3. Employee Utilization Rate
* **Name:** Unassigned Talent Ratio
* **Definition:** Calculated using Query Q7, which identifies employees not assigned to any project.
* **Current Value:** 15% (Example based on total employees vs unassigned).
* **Interpretation:** High values indicate that some employees' skills are not being utilized, suggesting a need for more projects or better task allocation.

## 4. Certification Growth
* **Name:** Workforce Technical Qualification
* **Definition:** Based on the new 'certifications' and 'employee_certifications' tables created in Q9.
* **Current Value:** 5 Certifications held by staff.
* **Interpretation:** Monitors the technical upskilling of the company, which increases the company's competitiveness in the market.
